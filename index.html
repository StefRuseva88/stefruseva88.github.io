My Projects

Here, I can showcase my passion for software testing and my love for creating innovative projects. One of the highlights is a project called PetZone.bg. This is an e-commerce platform focused on selling pet products. My main role was to thoroughly test the website to ensure that it functions correctly and provides a seamless user experience. This involved meticulous testing of various functionalities such as the product search feature, shopping cart, checkout process, and user account management.

In addition to functional testing, I also performed usability testing to identify any areas where the user experience could be improved. This included ensuring that the navigation was intuitive, the interface was user-friendly, and that the overall design met the needs of the target audience.

The test plan for this project outlines the testing strategy, objectives, resources, schedule, deliverables, and scope.
